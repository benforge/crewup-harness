## 风险

- 这个功能本质上允许任意客户端发起公开计数请求，存在被脚本刷高阅读量的业务风险，但不涉及用户隐私采集，也没有看到把访问者身份或设备信息写入统计链路。
- 迁移 `down` 使用 `ALTER TABLE ... DROP COLUMN`，在当前仓库语义下与 SQLite 迁移风格一致，回滚思路清晰；唯一的残余风险是如果底层运行环境切到更老的 SQLite 版本，`DROP COLUMN` 兼容性需要再确认。
