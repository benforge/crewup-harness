# 人工干预点

以下情况必须由人确认。

## 产品决策

- 是否进入开发。
- 优先级。
- 验收标准是否符合真实预期。
- 功能是否延期或拆分。
- 需求细化、设计方案和实施计划是否可以进入开发。
- release 后是否把 run 摘要沉淀到目标项目配置的长期文档目录。

## 技术决策

- 技术栈选择。
- 数据库类型。
- 部署平台。
- 认证方案。
- 付费云服务选择。

## 高风险操作

- 删除数据。
- 重置数据库。
- 生产环境部署。
- DNS 或域名变更。
- 密钥、Token、云服务账号配置。
- 破坏性 migration。

## 质量例外

- 测试未通过但仍要合并。
- 测试无法运行但仍要发布。
- Reviewer Agent 有阻塞问题但仍要继续。
