# Harness 经验沉淀

本文件用于把重复错误、流程漏洞和有效经验晋级为可维护的 harness 资产。

## 晋级规则

- 如果经验可以机械检查，晋级为 script 或 gate。
- 如果经验是行为约束，晋级为 `.harness/rules/`、`.harness/project/ai/rules/` 或本地 `.ai/rules.md`。
- 如果经验是项目结构知识，晋级为 dev-map 输入：package metadata 或本地 `.ai/rules.md`。
- 团队级规则不要只留在聊天记忆里。
