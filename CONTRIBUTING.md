# Contributing

Thanks for helping improve CrewUp.

## Before You Start

1. Run `npx crewup doctor`
2. Run `npx crewup check`
3. Run `npm run harness:check`

## Working Style

- Keep changes scoped.
- Prefer workflow clarity over extra abstraction.
- Update README when command behavior changes.
- Keep Markdown and YAML in UTF-8 without BOM.

## Commit Style

Use short, imperative commits.

Examples:

- `feat: add crewup doctor`
- `docs: clarify finish workflow`
- `fix: harden json parsing`
